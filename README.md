# 🏥 急诊抢救记录生成系统

## 概述

专为急诊科设计的抢救文书自动化工具。通过**点选式操作**快速记录急救过程中的用药、措施和管道信息，自动生成规范的抢救记录文本。

## 核心功能

### 📋 三大选择面板
| 面板 | 内容 | 项目数 |
|------|------|--------|
| 💊 急救药物 | 心肺复苏药、血管活性药、呼吸药、镇静镇痛药等 | 25+ |
| 🩺 抢救措施 | 循环支持、气道呼吸、诊疗操作、创伤处理 | 20+ |
| 🩸 管道通路 | 血管通路、气道管道、其他管道 | 12+ |

### ⏱ 时间线管理
- 每次点选**自动记录当前时间**
- 时间可**点击编辑**，精确到秒
- 按时间顺序自动排列
- 一键标记"抢救开始"/"抢救结束"

### 📄 自动生成记录文本
- 实时生成完整的抢救记录
- 包含基本信息、抢救经过、分类汇总、抢救结果
- 文本**完全可编辑**

### 📥 导出功能
- **复制文本**：一键复制到剪贴板
- **导出DOCX**：生成JSON数据 + Python脚本转Word文档
- **快捷键**：Ctrl+S 导出 / Ctrl+C 复制

## 使用方法

### 1. 打开工具
```bash
# 直接用浏览器打开
open emergency-rescue-doc/index.html

# 或双击 index.html 文件
```

### 2. 记录流程
1. 填写患者基本信息
2. 点击左侧标签切换"药物/措施/管道"
3. 点击具体项目 → 自动添加时间戳
4. 在右侧时间线查看和编辑
5. 点击"🟢 抢救开始"/"🔴 抢救结束"标记时间
6. 在底部编辑自动生成的记录文本

### 3. 导出DOCX文件
```bash
# 1. 在工具中点击"导出DOCX"获取JSON文件
# 2. 安装依赖
pip install python-docx

# 3. 运行导出脚本
python export_docx.py <导出的JSON文件路径>
```
